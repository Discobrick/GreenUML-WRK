Installation for Green 4.6.0:

System requirements:
- Eclipse version 4.6 (Neon)

- Open Eclipse, and navigate to Help > Install New Software...
- Click on Add... on the right hand side
- Fill the fields as following:
Name: GreenUML
Location: http://odo.lv/ftp/tools/GreenUpdateSite
- click OK

- At this point, two categories should be listed as available software sites: Green UML (required) and Green UML Add Ons. Choose what you want to install and click Next.
- Follow the dialogues and instructions, and Green will be installed!
- Restart Eclipse and everything should be functioning properly


Verifying the installation:
- Successful installation of Green can be checked by going to Help > Installation Details and typing "green".
Information about the installed plug-in should appear.
