Frequently Asked Questions

 

When I try to open a new diagram, Eclipse tells me that "The chosen operation is not currently available." What do I do?
In order for Green to run properly, it is necessary that you have Java 8 installed and that Java 8 is used to load the Green plugin.

How do I save my diagram as a GIF/JPEG/other image file?
In the 'Save As...' menu, you can choose on of four saving options: Green File, GIF, JPEG & PNG. In order to save in a particular formAat, you should append the appropriate extension to the filename.
Please Note: In order for the Green 'Save As...' menu option to be available, the Green editor window needs to be open.
